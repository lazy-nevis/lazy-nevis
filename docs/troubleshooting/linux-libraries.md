# Linux Libraries

Builds need WebKitGTK 4.1, GTK 3, Ayatana AppIndicator, librsvg, OpenSSL, ALSA, and packaging tools. Debian/Ubuntu development packages include `libwebkit2gtk-4.1-dev libayatana-appindicator3-dev librsvg2-dev libssl-dev libgtk-3-dev libasound2-dev patchelf`. Runtime names vary by distribution. AppImage may require FUSE 2 or extraction mode. Alpine/musl is unsupported. Report distribution/version, desktop, Wayland/X11, and package format.
