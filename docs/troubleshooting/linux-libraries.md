# Linux Libraries

Builds need WebKitGTK 4.1, GTK 3, Ayatana AppIndicator, librsvg, OpenSSL, ALSA, PipeWire/GBM/EGL/Wayland (for `xcap` screen capture on Linux), Clang/libclang (bindgen), and packaging tools. Debian/Ubuntu development packages include `libwebkit2gtk-4.1-dev libayatana-appindicator3-dev librsvg2-dev libssl-dev libgtk-3-dev libasound2-dev libpipewire-0.3-dev libgbm-dev libegl-dev libwayland-dev clang libclang-dev patchelf`. Runtime names vary by distribution. AppImage may require FUSE 2 or extraction mode. Alpine/musl is unsupported. Report distribution/version, desktop, Wayland/X11, and package format.
